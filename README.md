# Keds

A prototype for a generic CLI plugin framework.

# Developer Setup

		$ brew install dep
		$ brew install protobuf
		$ make go-tools
		$ make test

# Running

		$ go run main.go

# Debugging

		$ open http://localhost:8081/debug/requests
		$ open http://localhost:8081/debug/events
